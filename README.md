# Flavos BioMap

*O rastro da floresta* 
"Viver e conviver com a sociobiodiversidade como caminho para o futuro"

Um projeto React Native (Expo) interativo planejado com foco na valorização da origens socioculturais dos biomas brasileiros.

## Objetivo
Aplicativo conectado a uma exposição cultural contendo produtos físicos vinculados aos biomas e saberes tradicionais do Brasil. O app funciona como ponte tecnológica (via QR Code) entre as pessoas e a rastreabilidade da origem dos produtos na natureza.

## Estrutura do Repositório Único
**Atenção:** Visando agilidade para MVP, redução de complexidade de CI/CD para uma amostra e capacidade fácil de compilação multi-plataforma via Expo e web (Vercel), este projeto foi configurado intencionalmente como um **Repositório Único** (`Single Repository`).
**NÃO** faz uso de abordagens como Monorepo ou Turborepo, assegurando um baseline leve e profissional.

### Tecnologias:
- React Native + Expo (SDK ~50)
- TypeScript
- Supabase (Autenticação / DB PostgreSQL / Realtime)
- React Navigation (Navegação Nativa)
- expo-barcode-scanner e react-native-maps

### Referência Visual
- Tons de Verde, Terra, Bege, Branco Suave
- Natureza, floresta, humano.
- Minimalismo Limpo.
